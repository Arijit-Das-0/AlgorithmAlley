Rightmost Different Bit

Problem Statement

Given two numbers M and N, find the position of the rightmost different bit in their binary representation. If both M and N are the same, then return -1.

Example 1:

Input: 
M = 11, N = 9
Output: 
2
Explanation: 
Binary representation of the given numbers are: 1011 and 1001, 2nd bit from the right is different.

Example 2:

Input: 
M = 52, N = 4
Output: 
5
Explanation: 
Binary representation of the given numbers are: 110100 and 0100, 5th-bit from the right is different.

Code

                
int posOfRightMostDiffBit(int m, int n)
{
    if (m == n)
        return -1;
        
    int res = m ^ n;
    res = res & (-res);
    return log2(res) + 1;
}

Complexity

Time Complexity: O(max(log M, log N))

Auxiliary Space: O(1)

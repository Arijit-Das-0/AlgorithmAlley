Reach the Nth Point

Problem Statement

There are N points on the road. You can step ahead by 1 or 2. Starting from point 0 and moving to point i+1 after taking a step of length one, find the number of ways you can reach point N.

Example 1:

Input:
N = 4
Output:
5
Explanation:
Three ways to reach 4th point:
{1, 1, 1, 1}, {1, 1, 2}, {1, 2, 1}, {2, 1, 1}, {2, 2}

Example 2:

Input:
N = 5
Output:
8
Explanation:
Eight ways to reach 5th point:
{1, 1, 1, 1, 1}, {1, 1, 1, 2}, {1, 1, 2, 1}, {1, 2, 1, 1}, {2, 1, 1, 1}, {1, 2, 2}, {2, 1, 2}, {2, 2, 1}

Code

                
int nthPoint(int n){
    int first = 1;
    int second = 2;
    int mod = 1e9+7;
    if(n == 1) return 1;
    for(int i=2; i <n ; i++){
        int sum = (first+second)%mod;
        first = second;
        second = sum;
    }
    return second;
}

Complexity

Time Complexity: O(N)

Space Complexity: O(1)

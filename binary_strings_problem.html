Count Binary Strings without Consecutive 1's

Problem Statement

Given a positive integer N, count all possible distinct binary strings of length N such that there are no consecutive 1â€™s. Output your answer modulo 109 + 7.

Example:

Input: N = 3

Output: 5

                Explanation:
                5 strings are (000, 001, 010, 100, 101).

Code

                
                    ll countStrings(int n) {
                        long long first = 1;
                        long long second = 2;
                        long long mod = 1e9+7;
                        for(int i=2; i<=n; i++){
                            long long sum = (first+second)%mod;
                            first = second;
                            second = sum;
                        }
                        return second;
                    }

Complexity

Time Complexity: O(N)

Space Complexity: O(N)

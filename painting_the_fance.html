Painting the Fence

Problem Statement

Given a fence with n posts and k colors, find out the number of ways of painting the fence so that not more than two consecutive posts have the same colors.

Input:
n = 3
k = 2

Example 2:

Input:
n = 2
k = 4 
Output: 16
Explanation: 
After coloring the first post with 4 possible combinations, you can still color the next posts with all 4 colors.
Total possible combinations will be 4x4=16.

Code

                
                    long long countWays(int n, int k){
                        // There are k ways to color first post
                        long long total = k;
                        int mod = 1000000007;
                        // There are 0 ways for single post to
                        // violate (same color) and k ways to
                        // not violate (different color)
                        long long same = 0, diff = k;
                     
                        // Fill for 2 posts onwards
                        for (int i = 2; i <= n; i++) {
                            // Current same is same as previous diff
                            same = diff;
                            // We always have k-1 choices for next post
                            diff = total * (k - 1)% mod;
                            // Total choices till i.
                            total = (same + diff) % mod;
                        }
                        return total;
                    }

Complexity

Time Complexity: O(N)

Auxiliary Space: O(N)

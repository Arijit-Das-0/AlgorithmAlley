Count More than N/K Occurrences

Problem Statement

Given an array arr of size N and an element k. Find the count of elements in the array that appear more than n/k times.

Example 1:

Input:
N = 8
arr = [3,1,2,2,1,2,3,3]
k = 4
Output: 
2
Explanation: 
In the given array, 3 and 2 are the only elements that appear more than n/k times.

Example 2:

Input:
N = 4
arr = [2,3,3,2]
k = 3
Output: 
2
Explanation: 
In the given array, 3 and 2 are the only elements that appear more than n/k times. So the count of elements are 2.
